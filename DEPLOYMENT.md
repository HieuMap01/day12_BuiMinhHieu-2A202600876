  }

### API Test Without Authentication

```powershell
$body = @{ question = "Hello" } | ConvertTo-Json

Invoke-RestMethod `
  -Method Post `
  -Uri "https://day12-agent.onrender.com/ask" `
  -ContentType "application/json" `
  -Body $body
```

Expected:

```text
401 Unauthorized
```

### API Test With Authentication

Set `AGENT_API_KEY` in Render Environment to the same value used in the request.

```powershell
$body = @{ question = "Hello" } | ConvertTo-Json

Invoke-RestMethod `
  -Method Post `
  -Uri "https://day12-agent.onrender.com/ask" `
  -Headers @{ "X-API-Key" = "my-secret-key" } `
  -ContentType "application/json" `
  -Body $body
```

Expected:

```text
200 OK with an agent answer
```

If this returns `401`, redeploy after saving the Render environment variable:

```text
AGENT_API_KEY=my-secret-key
```
